# RestApi-Django-with-postgresql

This api build using django with jwt authentication.
Postgresql Database include

Please Configure your database in settings.py
DATABASES = {

    'default': {

        'ENGINE': 'django.db.backends.<databse>',

        'NAME': '<database name>',

        'USER': '<username>',

        'PASSWORD': '<password>',

        'HOST': '<host>',

        'PORT': '<post no>',

    }

}
